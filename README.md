# Hang TEN - The Game of Hangman

Hang TEN is a simple yet entertaining game of hangman/hangTEN that you can play right from your browser.

How to Play
1. Click the "Start Game" button to begin the game.
2. Guess the letters to uncover the word.
3. You have 6 attempts to guess the word before the our stick friend is "hanging ten" and surfing drunk which we dont want!
4. Click the "Clue" button for a hint about the word.
5. Click the "Hint" button to see the part of speech of the word.

## How to Win
If you guess all the letters correctly before the surfer stick is hung, you win the game. But, if the surfer stick is hung before you guess the word, then you lose the game.

## Technical Details
The game is built using HTML, CSS, and JavaScript. It fetches a random word from a third-party API and uses it for the game. It also uses another third-party API to provide hints and part of speech for the selected word. The game is designed to be responsive and can be played on any device.

To play the game, simply download or clone the repository and open the index.html file in your browser.

Please note this game is currently being added to and worked on. 

Have fun playing Hang TEN!
